# Ignis Tuus

> **Ignis Tuus, Umbra Nostra** — *Your fire, our shadow*

## About Us

**Ignis Tuus** is an organization specializing in **UAV (Unmanned Aerial Vehicle) detection**. We develop cutting-edge solutions to identify, track, and neutralize unauthorized drone activity across a wide range of environments.

## What We Do

- 🛸 **UAV Detection** — Advanced systems for detecting unmanned aerial vehicles in real time
- 📡 **Signal Intelligence** — RF and radar-based identification of drone communications and control links
- 🧠 **AI-Powered Analysis** — Machine learning models for classifying and tracking aerial threats
- 🔒 **Security Integration** — Seamless integration with existing perimeter security and C2 systems

## Our Mission

We operate in the shadows so that you can stand in the light. Wherever there is a threat from above, Ignis Tuus provides the awareness and response capability to keep people and assets safe.
